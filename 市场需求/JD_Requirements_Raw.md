# AI Agent 相关职位 JD 原始信息

## JD 1: 核心要求 (Hacker 精神与工程能力)
**希望你是：**
1. **强烈的 Hacker 精神**：不纸上谈兵，真正动手把想法落地：Prompt 工程能力是你的第二天性，懂得如何与 LLM 打交道；
2. **对 AI 工具生态极度熟悉**（如 Claude、Cursor、Devin 等），善于用 AI 放大效率；
3. **有大型基础设施经验**，了解多租户架构；
4. **维护过工程级代码**；
5. **熟悉 TypeScript**，具备前后端协作经验；
6. **能独立推进项目**，无需 Tech Lead/CTO 进行细节指导；能在会议 / Slack / 设计文档 / 日报等形式中清晰表达技术方案与工程判断。

---

## JD 2: 职位要求 (技术栈与框架)
**职位要求：**
1. **精通 Python、Go 编程语言**，具备良好的软件框架和应用设计能力；
2. **具备丰富的 Prompt Engineering 经验**和大模型应用开发经验；
3. **熟悉 RAG 技术实现原理**，能够设计和优化知识检索系统；
4. **深入理解 AI Agent 技术栈**，熟练掌握主流智能体开发框架，具备 LangChain、LlamaIndex 等框架的实战经验，能够构建可扩展的智能体系统；
5. **熟悉 MCP、HTTP、WebSocket 协议**，具备 Client-Server 架构的开发与集成能力，能够设计高效的 Agent 通信机制；
6. **熟悉 Llama、AutoGen、CrewAI、Dify、LangGraph、LangFlow 等 Agent 框架平台**的应用；
7. **具备 LLM 应用的部署和运维能力**，确保系统稳定性和可用性。

---

## JD 3: Agent 产品负责人
**核心职责 (What You'll Do):**
1. **Agent 技能定义 (Skill Design)**:
   - 深入挖掘企业内部场景（如销售、客服、研发），定义 Agent 框架。
   - 撰写高质量的 System Prompts 和 User Stories，将模糊的业务需求转化为 AI 指令集。
2. **编排与工作流 (Workflow Orchestration)**:
   - 设计多步任务执行逻辑（SOP）。
   - 协作工程团队将 ReAct 或 LangGraph 思想规划 Agent 的思考路径。
3. **评测与迭代 (Eval & Iteration)**:
   - 建立产品的“智商标准”。设计测试集 (Golden Dataset)，量化评估 Agent 的任务完成率和幻觉率。
   - 通过分析用户对话日志 (Bad cases)，持续迭代 Prompt 和知识库 (RAG)，提升 Agent 的准确度。
4. **人机交互创新 (HCI)**:
   - 探索 Slack/Teams/Web 中的新型交互形态。

**任职要求 (Requirements):**
1. **AI Native 认知**：不仅玩过各类 AI 工具，还尝试过构建 Workflow (Coze/Dify/GPTs)。深刻理解 Context Window、Temperature、Function Calling 等技术概念对产品体验的影响。
2. **Prompt Engineering 能力**：能自己写出复杂的 Prompt 来验证想法，而不是等着工程师来调教模型。
3. **极强的逻辑思维**：将逻辑和审美融入整个业务体系。

**加分项 (Bonus):**
1. 有实际的 AI Agent / 客服机器人 / RPA 产品落地经验。
2. 熟悉 LangChain / LangSmith 等主流 AI 开发框架的概念。
3. 有代码能力 (Python/JS)，能自己写简单的脚本跑通 API。

## JD 4: AI Agent 开发工程师 (全生命周期)
**工作职责：**
1. 负责 AI Agent 系统的**全生命周期开发工作**，涵盖从架构设计，到详细的工作流搭建，全面的测试优化以及最终的上线部署；
2. 深入探索 LLM、RAG、MCP 等前沿技术，研究与现有系统的融合方案；
3. 紧密跟踪 AI 技术前沿动态，对现有系统进行持续优化；
4. **主导核心功能模块开发**：多 Agent 协作机制、精准意图识别、高效记忆管理、强大知识库及灵活工具调用；
5. **监控与评估体系**：实时监测运行数据，持续优化系统性能，确保稳定可靠。

**任职要求：**
1. 本科及以上学历，计算机科学、人工智能等相关专业；
2. **精通 Go 或 Python** 及 AI 应用开发框架（LangChain, LangGraph），有 Coze, Dify 等低代码平台经验；
3. **深入理解 RAG 技术**，具备向量数据库实践经验；
4. **熟练掌握 Prompt 工程**，能对复杂问题进行合理拆解和规划；
5. **熟悉多 Agent 协作架构**与任务调度逻辑；
6. 有微调经验优先。

---

## JD 5: 大模型算法/Agent 工程师
**职位描述：**
1. **构建 Agent 核心逻辑**：任务规划、编排、工具调用、MCP、Skill、多轮对话、记忆、RAG、上下文工程、多智能体协同；
2. **业务与商用场景搭建**：提供生产级智能助手；
3. **长短期记忆架构**：解决上下文遗忘与注意力分散，提升交互一致性；
4. **RAG 系统构建**：检索、召回、文本处理等模块优化；
5. **工具开发**：接入搜索、浏览器、虚拟机等外部 API、MCP 和 Skills；
6. **Prompt 设计与调优**；
7. **科学评测方案**：设计可量化指标和测试方案。

**任职要求：**
1. AI、计算机、数学等专业**硕士及以上学历**；
2. 良好编程能力，**精通 PyTorch** 等训练框架；
3. **熟悉大模型训练/推理原理**：提示词工程、评测、模型部署；
4. **强自驱力**，推动技术从研究到落地的全链路闭环；
5. 有 LangGraph, Dify 等 Agent 框架开发经验；
6. **强工程能力**，熟悉 Cursor, Claude Code 等 AI 工具。
