# AI Agent 相关职位需求总结报告

通过对多份 JD 的分析，当前市场对 AI Agent 相关人才的需求高度集中在**工程落地能力**、**底层协议理解**以及**复杂工作流编排**三个维度。以下是核心要求的深度总结：

## 1. 核心技术栈 (Tech Stack)
*   **编程语言**：
    *   **Python**: 绝对主流，要求精通及其生态。
    *   **Go**: 常用于高性能后端服务、基础设施开发。
    *   **TypeScript/JS**: 用于前端交互、全栈协作或特定脚本编写。
*   **AI 框架与平台**：
    *   **开发框架**: LangChain, **LangGraph** (高频提及，强调复杂工作流), LlamaIndex。
    *   **低代码平台**: **Dify**, **Coze** (强调快速原型与业务落地)。
    *   **多智能体框架**: AutoGen, CrewAI。
*   **大模型底层**：
    *   **PyTorch**: 算法岗位要求精通，涉及微调与推理优化。
    *   **协议**: **MCP (Model Context Protocol)** (新兴热点), HTTP, WebSocket。

## 2. 关键能力模块 (Key Competencies)
*   **Prompt Engineering (提示词工程)**：
    *   不再局限于简单提问，要求能设计复杂指令集、System Prompts，并进行自动化调优。
*   **RAG (检索增强生成)**：
    *   深入理解向量数据库，能优化检索、召回、重排序全链路。
*   **Agent 核心逻辑**：
    *   **任务编排**: SOP 设计、ReAct 思想、多步逻辑规划。
    *   **记忆管理**: 长短期记忆架构，解决上下文遗忘问题。
    *   **工具调用**: API 集成、Function Calling、外部工具 (浏览器/虚拟机) 接入。
*   **评测与监控**：
    *   建立“智商标准”，使用 Golden Dataset 进行量化评估，解决幻觉问题。

## 3. 岗位职责共性 (Common Responsibilities)
*   **全生命周期开发**：从架构设计、工作流搭建到测试部署、持续迭代。
*   **场景深度挖掘**：将企业内部业务场景（客服、研发、销售）转化为 Agent 解决方案。
*   **前沿技术追踪**：紧跟 LLM/Agent 领域动态，快速将新技术（如 MCP）引入生产。

## 4. 软素质与加分项 (Soft Skills & Bonus)
*   **Hacker 精神**：强烈的动手能力，拒绝纸上谈兵，善用 AI 工具 (Cursor, Claude Code, Devin) 提升效率。
*   **工程化思维**：维护过工程级代码，了解多租户架构、基础设施运维。
*   **AI Native 认知**：对 Context Window、Temperature 等超参数有深刻的直觉。
*   **独立性**：能独立推进项目，清晰表达技术方案。

## 5. 趋势观察
1.  **从“对话”转向“协作”**：多智能体 (Multi-Agent) 协同处理复杂任务成为标配要求。
2.  **协议标准化**：**MCP** 的提及频率显著上升，显示出行业对 Agent 与工具交互标准化的重视。
3.  **强调闭环评测**：企业对 Agent 的要求从“能跑通”转向“稳定、可量化、低幻觉”。
